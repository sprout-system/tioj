# Contest 題目一次性加減 Tag

## 操作步驟

1. 進入要管理的比賽頁面（`/contests/[contest-id]`）。
2. 在題目列表上方的輸入框輸入要操作的標籤名稱。
3. 按下 **Add Tag to All Tasks** 或 **Remove Tag from All Tasks** 把這個標籤加到每個題目或從每個題目移除。

## 注意事項

- 會套用到比賽中的所有題目；要只挑其中幾題的話要到題目的個別頁面。
- 標籤名稱前後的空白會自動去除（例如 ` graph ` 會存成 `graph`）。
- 加上不存在的標籤時會自動創建該標籤，所以如果想要統一的話要確定是不是你要的。
- 移除標籤時，若題目本來就沒有這個標籤的話會直接跳過。
- 每次只能處理一個標籤名稱，要操作多個要重複上述步驟。
